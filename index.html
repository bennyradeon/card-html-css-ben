Template visual yang bisa dipilih siswa (versi fix preview VSCode)

Contoh di bawah dibuat mengikuti desain pada gambar yang Anda unggah: avatar lingkaran, teks besar, badge di kanan atas, daftar portofolio, tombol hijau, label indikator di kiri bawah, serta variasi glow dan progress bar. Versi ini sudah diperbaiki agar lebih stabil saat dibuka di preview VSCode maupun browser biasa.

Panduan cepat untuk siswa

1. Edit isi teks Ganti nama, kelas, deskripsi, portofolio, dan label badge.
2. Sesuaikan layout Pilih layout foto kiri, header strip, dua tombol, atau daftar skill sesuai kode tugas. Buka dengan browser atau Live Server agar preview lebih akurat.
3. Tambah bonus Badge untuk X, glow effect untuk Y, dan progress bar untuk Z. Semua contoh tetap satu file agar mudah dicoba.
4. Kreasikan tema Ubah warna hijau, biru, oranye, ungu, coklat, atau abu-abu berdasarkan tema A–F.
