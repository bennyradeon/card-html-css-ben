Tujuan Pembelajaran

  • Peserta didik memahami struktur dasar HTML (<head>, <body>, heading, paragraf, gambar, tautan).
  • Peserta didik mampu menerapkan CSS dasar untuk layout: display, justify-content, padding, margin, border, border-radius, dan box-shadow.
  • Peserta didik mampu membuat tampilan kartu profil yang rapi, modern, dan responsif sederhana.
  • Peserta didik mampu membuat variasi desain secara mandiri (kreatif) tanpa menyalin kode teman.

Prasyarat & File

File yang diperlukan

  • index.html — berisi struktur HTML.
  • CSS ditulis di dalam tag <style> pada file index.html.
  • (Opsional) foto.jpg — jika memakai foto lokal. Jika tidak, gunakan URL gambar.

Aturan praktik

  • Tuliskan kode dengan rapi (indentasi).
  • Setiap perubahan: simpan file → refresh browser.
  • Gunakan komentar seperlunya untuk menandai bagian penting.

Demo Hasil

Silakan klik tombol preview untuk melihat hasil pada setiap tahap, lalu cocokkan dengan praktik kalian.

Catatan: Preview di modul ini merender kode contoh (tidak membaca file di komputer Anda). Saat praktik, Anda tetap menulis di file index.html dan tag <style> di index.html.

Tahapan Praktik (Berurutan)

Kerjakan tahap 1 sampai 8 secara berurutan. Setiap tahap menambahkan fitur baru. Bagian baru pada kode ditandai dengan warna highlight.

Cara menyalin kode: gunakan tombol Copy Kode Bersih pada setiap blok kode. Tombol ini akan menyalin kode tanpa highlight, sehingga aman ditempel di file siswa.

Tahap 1 — Struktur HTML Dasar

Fokus: Membuat kerangka HTML, judul, paragraf, dan wadah div.card.

Satu file saja: pada tahap ini CSS ditulis di dalam <style> di bagian <head>. Jadi kalian cukup membuat index.html.
Catatan: Mulai tahap ini, kita sudah menyiapkan CSS dasar untuk membuat tampilan kartu terlihat jelas (background, padding, radius, shadow).
Wadah kartu
<div class="card">
  ...isi kartu...
</div>
Class card dipakai untuk memberi style lewat CSS.
Heading
<h1>Judul</h1>
Gunakan h1 untuk judul utama.
Paragraf
<p>Kalimat...</p>
Gunakan p untuk teks deskripsi.
Catatan penting: Kalau kalian hanya menyalin HTML, kartu bisa terlihat seperti teks biasa. Pastikan CSS .card (Tahap 5) juga dibuat agar tampilan kartu muncul dengan benar.

Kode index.html (Tahap 1)

Legenda: Baris baru pada tahap ini
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Profile Card</title>
  <style>
body{
  margin: 0;
  min-height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  padding: 24px;
  background-color: #f7f7f7;
  font-family: system-ui, -apple-system, sans-serif;
}
.card{
  width: 320px;
  background: rgb(209, 255, 149);
  padding: 24px;
  border-radius: 16px;
  text-align: center;
  box-shadow: 0 8px 15px rgba(0,0,0,0.10);
}
</style>
</head>

<body>
  <div class="card">
    <h1>HELLO! SAYA SISWA XI</h1>

Tahap 5 — Menambahkan CSS Dasar (Body & Card)

Fokus: Menambahkan <style> lalu mengatur tampilan body dan .card (warna, ukuran, padding, radius, shadow).

Satu file saja: pada tahap ini CSS ditulis di dalam <style> di bagian <head>. Jadi kalian cukup membuat index.html.
Catatan: Kode CSS pada tahap ini melanjutkan Tahap 4. Tambahan baru: merapikan ukuran judul (h1) dan jarak paragraf (p).

Selector yang digunakan

  • body { ... } → untuk seluruh halaman.
  • .card { ... } → untuk elemen yang memiliki class="card".

Box Model (wajib paham)

PropertiArtiContoh
paddingJarak dalampadding: 25px;
marginJarak luarmargin-top: 50px;
border-radiusSudut membulat15px
box-shadowBayangan0 8px 15px ...

Contoh penulisan CSS (inti)

Memusatkan kartu
body{
  display: flex;
  justify-content: center;
  padding-top: 50px;
}
Flexbox membuat elemen anak (card) mudah dipusatkan.
Styling card
.card{
  width: 280px;
  padding: 25px;
  border-radius: 15px;
  box-shadow: 0 8px 15px rgba(0,0,0,0.1);
}
Radius + shadow membuat tampilan modern.
Tipografi
body{
  font-family: system-ui, -apple-system, sans-serif;
}
Font system-ui terlihat modern dan nyaman dibaca.

Tahap 7 — Menjadikan Link Seperti Tombol

Fokus: Mengatur tag <a> agar tampak seperti tombol dengan padding, background, border-radius, dan hover.

Satu file saja: pada tahap ini CSS ditulis di dalam <style> di bagian <head>. Jadi kalian cukup membuat index.html.
Catatan: Kode CSS tetap melanjutkan tahap sebelumnya. Tambahan baru: efek hover pada .card agar terasa lebih modern.

Konsep penting

  • display: inline-block agar padding bekerja seperti tombol.
  • text-decoration: none untuk menghilangkan garis bawah link.
  • a:hover untuk efek saat mouse diarahkan.
Contoh CSS tombol
a{
  display:inline-block;
  padding:10px 20px;
  background:#333;
  color:white;
  text-decoration:none;
  border-radius:8px;
}
Nilai padding boleh disesuaikan sesuai kebutuhan.

Hover & transisi

Transisi
a{ transition: 0.3s; }
Efek berubah halus, tidak “patah”.
Hover
a:hover{
  background:#555;
}
Saat mouse di atas link, warna berubah.
Scale
a:hover{
  transform: scale(1.05);
}
Memberi efek tombol sedikit membesar.

Tahap 8 — Pengembangan Keren (Variasi Tema + Responsif Ringan)

Fokus: Membuat tampilan lebih modern dengan gradient halus, border lebih rapi, dan responsif sederhana.

Satu file saja: pada tahap ini CSS ditulis di dalam <style> di bagian <head>. Jadi kalian cukup membuat index.html.
Catatan: Kode CSS tetap melanjutkan tahap sebelumnya. Tambahan baru: background halaman dibuat lebih modern dengan gradient.

Ide pengembangan

  • Ubah background card menjadi gradient.
  • Tambahkan max-width agar aman di layar kecil.
  • Rapikan garis <hr> melalui CSS (bukan inline style).

Contoh responsif ringan

.card{ width: min(320px, calc(100% - 24px)); }
Card akan menyesuaikan layar HP tanpa pecah.

Kode index.html (Tahap 8)

Legenda: Baris baru pada tahap ini
<!DOCTYPE html>
<html lang="id">

Latihan & Soal (Pemanasan)

Latihan Cepat (5–10 menit)

  1. Ubah teks <h1> menjadi nama dan kelas Anda.
  2. Ubah background pada .card menjadi warna lain (mis. biru muda).
  3. Ubah ukuran foto menjadi 140x140.
  4. Ubah teks tombol menjadi “Kunjungi Portofolio”.
  5. Ubah efek hover tombol menjadi lebih halus (mis. scale(1.02)).

Soal Pemahaman (Jawab Singkat)

  1. Apa fungsi atribut alt pada tag <img>?
  2. Mengapa display: inline-block berguna saat membuat link menjadi tombol?
  3. Apa perbedaan padding dan margin?
  4. Apa fungsi box-shadow pada card?
  5. Apa fungsi a:hover?

Soal Praktik Mudah (Dikerjakan di Lab)

Buat variasi dari kartu profil dengan ketentuan berikut:

  • Tambahkan satu teks baru: “Hobi Saya:” lalu isi hobinya.
  • Tambahkan satu garis pemisah lagi (<hr>) sebelum tombol.
  • Warna tombol berbeda dari contoh, tetapi tetap kontras dan terbaca.
  • Tambahkan efek hover pada gambar (misalnya transform: scale(1.03)) dengan transisi.

Checklist Wajib (Dinilai)

  • Halaman tidak melebar (tidak perlu geser kanan-kiri).
  • Card berada di tengah (gunakan display:flex atau display:grid).
  • Ada judul (h1), foto (img), deskripsi (p), dan subbagian (mis. h3).
  • Tautan dibuat seperti tombol (CSS pada a + :hover).
  • Minimal 8 properti CSS digunakan dengan benar.
  • Variasi sesuai kode tantangan (warna + layout + bonus).

Instruksi Umum (Wajib)

  1. Buat 1 halaman index.html (boleh internal CSS di <style> atau eksternal tag <style> di index.html).
  2. Tampilkan satu kartu utama di tengah halaman (seperti materi).
  3. Kartu wajib berisi: Judul, Foto, Deskripsi, Bagian “Tentang”, dan Tombol link.
  4. Gunakan minimal 8 properti CSS dari materi (mis. flex, padding, radius, shadow, transition, hover).

Variasi Anti-Copy (Setiap Siswa Berbeda)

Setiap siswa wajib memilih 1 Tema + 1 Layout Variasi + 1 Komponen Bonus dari daftar berikut.

Tema (pilih 1)
  • Tema “Olahraga”
  • Tema “Musik”
  • Tema “Kuliner”
  • Tema “Game”
  • Tema “Literasi/Book”
  • Tema “Teknologi”
Layout Variasi (pilih 1)
  • Foto di kiri, teks di kanan (gunakan flex).
  • Card memiliki header warna di atas (strip).
  • Card memiliki 2 tombol: “Profil” & “Kontak”.
  • Card memiliki daftar 3 skill (pakai <ul>).
Komponen Bonus (pilih 1)
  • Tambahkan badge (mis. “XI RPL 1”).
  • Tambahkan ikon sederhana (boleh pakai karakter).
  • Tambahkan progress bar skill (CSS sederhana).
  • Tambahkan efek hover pada card (mis. sedikit naik).

Paket Tantangan (Agar Unik)

Bapak/Ibu Guru akan membagikan Kode Tantangan per siswa. Kode ini menentukan kombinasi warna dan komponen yang wajib ada. Peserta didik mengerjakan sesuai kode masing-masing.

Generator Kode Tantangan (untuk contoh)
—
Guru dapat menuliskan kode berbeda untuk setiap siswa (mis. di papan). Siswa mengikuti aturan kode tersebut.

Cara membaca kode

  • Warna: Kode berawalan A/B/C/D menentukan warna card (guru sediakan palet).
  • Layout: Angka 1–4 menentukan variasi layout.
  • Bonus: Huruf X/Y/Z menentukan komponen bonus.
BagianOpsiContoh
WarnaA=Hijau, B=Biru, C=Ungu, D=OranyeA2Y
Layout1=Foto kiri, 2=Header strip, 3=2 tombol, 4=Daftar skillA2Y → layout 2
BonusX=Badge, Y=Hover card, Z=Progress skillA2Y → bonus Y
Catatan pelaksanaan: Kalian tetap memakai konsep kartu profil, tetapi kombinasinya dibuat berbeda agar hasil karya mandiri. Dengan demikian, hasil akhir akan bervariasi dan lebih kreatif.
